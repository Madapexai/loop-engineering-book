# 别调Prompt了，去写Loop

> AI Agent时代的软件工程新范式
> 作者：**Yason** · MindApex · VokoForge

## 简介

这不是一本概念科普书。市面上已经有太多文章告诉你"Loop Engineering是什么"。你不需要花几百页来理解一个定义。

这本书要做的，是带你从"知道"走向"做到"。

- **框架**：六大核心模块如何组合成一个完整的Loop系统
- **方法论**：从零设计一个Loop的五步流程
- **避坑指南**：六大失败模式和三大深层风险，以及如何规避
- **真实案例**：从Ralph Loop到明略科技，从字节跳动到Amplitude
- **M-LOOP框架**：1个Rex中枢 + 4条闭环 + 7个门禁的实战体系
